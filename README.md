## Project Description
The project focuses on detecting and counting vehicles in images and videos using **YOLOv8** and **OpenCV**.  
It demonstrates the application of computer vision techniques for object detection and counting.

## Features
- Vehicle detection in images
- Vehicle detection and counting in videos
- Estimation of total vehicles
- Supports multiple vehicle classes

## Technologies Used
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- NumPy
- Pillow

## Vehicle Classes
- Car
- Truck
- Bus
- Motorbike
- Bicycle
